# Malta Admin Panel

Vite + React kullanılarak geliştirilmiş admin panel.

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Sunucuyu başlatın:
```bash
npm run dev
```

Admin panel `http://localhost:5173` adresinde çalışacaktır.

## Özellikler

- **Dashboard**: Form istatistikleri ve hızlı erişim
- **Form Yönetimi**: Tüm form kayıtlarını görüntüleme ve silme
- **SEO Yönetimi**: Sayfa bazlı SEO ayarlarını yönetme

## Giriş Bilgileri

- Kullanıcı Adı: `admin`
